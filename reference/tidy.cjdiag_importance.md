# Tidy a cjdiag_importance object

Tidy a cjdiag_importance object

## Usage

``` r
tidy.cjdiag_importance(x, ...)
```

## Arguments

- x:

  An importance object from
  [`importance()`](https://dkarpa.github.io/cjdiag/reference/importance.md)

- ...:

  Unused

## Value

A tibble of importance results
